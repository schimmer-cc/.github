# schimmer.cc

> A Minecraft server network, built from the ground up.

**schimmer.cc** is a personal project by [Fiereu](https://github.com/Fiereu) — an ambitious undertaking to design, develop, and operate a fully custom Minecraft server network completely from scratch.

---

## What is this?

This repository is the public face of the schimmer.cc project. Rather than relying on off-the-shelf server software and pre-built plugins, schimmer.cc is an exercise in building things the hard way — crafting custom infrastructure, game logic, and tooling to power a complete network experience.

Everything from server architecture to player-facing features is designed and implemented by Fiereu, with a focus on quality, performance, and learning through doing.

---

## Goals

- Build a fully functional Minecraft server network with custom-developed components
- Design scalable backend infrastructure suited for multiplayer game servers
- Implement original gameplay systems and features
- Document the journey and the decisions made along the way

---

## Project Status

🚧 **Work in progress** — schimmer.cc is actively under development. Things will break, change, and (hopefully) improve over time.
